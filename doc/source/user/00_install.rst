Installation
============

\*nix distributions
-------------------

Use your package manager, there are packages for Debian (and probably derivatives) and Arch via AUR.


Python virtualenv
-----------------

.. code:: bash

    # create venv
    python -m venv mpd_sima-venv
    # activate
    . ./mpd_sima-venv/bin/activate
    # Install the application
    pip install MPD_sima
    # Print help message
    mpd-sima --help
